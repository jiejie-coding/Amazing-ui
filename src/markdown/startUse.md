  # 开始

  请先 [安装](#/doc/install) Amazing UI

  然后根据下面的代码导入需要的组件

```js
import {Button, Tabs, Switch, Dialog} from "amazing-ui-233"
```

  这允许您使用我提供的组件。

# Vue components

代码示例：

```html
<template>
  <div>
    <Button>按钮</Button>
  </div>
</template>
<script>
  import {Button, Tabs, Switch, Dialog} from "king-ui"
  export default {
    components: {Button}
  }
</script>
```
